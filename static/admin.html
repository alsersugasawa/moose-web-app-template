Developer Tools

Scaffolding, exports, and integration resources

TypeScript Client

Download an auto-generated TypeScript SDK derived from the live OpenAPI specification. The file contains typed interfaces for all schemas and async fetch wrapper functions for every endpoint.

Download client.ts View OpenAPI JSON

Scaffold CLI

Generate boilerplate for new resources from the command line. Run from the project root:

python -m scaffold router <name>

This creates a stub router in app/routers/, appends schema classes to app/schemas.py, and generates a SQL migration stub in migrations/.

Plugin Architecture

Add functionality without modifying core files. Place a Python package in the plugins/ directory — any router attribute is registered automatically on startup.

plugins/
  my_feature/
    __init__.py    # export router
    router.py      # FastAPI APIRouter

An example plugin is available at plugins/example/. Test it: GET /api/plugins/example/ping
